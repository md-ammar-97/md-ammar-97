## 🤖 How I Build

> I use AI-assisted development to rapidly prototype product ideas, validate workflows, and communicate with engineering teams.

My strength is not writing perfect production code from scratch — it is:

- **Discovering** — stakeholder interviews, problem statements, opportunity sizing before any build
- **Designing** — MVP scope, user journeys, business rules, and acceptance criteria that engineering can execute
- **Prototyping** — working product in days using Claude Code, Cursor, Lovable, and modern stacks
- **Testing** — real users, real feedback, fast iteration cycles
- **Shipping** — handing off with full PRDs and specs, or deploying directly when the prototype is the product
